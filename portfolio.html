Project Overview

This is a full stack project designed to showcase a developer portfolio. It includes a visually appealing frontend, a backend built using Node.js, and features such as form submissions, dynamic project listings, and responsive design.

Technologies used include HTML, CSS (with dark theme), JavaScript, Node.js, and Express. Contact form data is handled securely on the server, and project pages like this one are dynamically added or updated.

  • Interactive UI with particle background
  • Responsive design for all devices
  • Form data routed to backend using Node.js
  • Expandable project pages with “View More”

Start Date: May 2025

Status: Currently in development

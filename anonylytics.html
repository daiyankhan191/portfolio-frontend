Project Overview

Anonylytics: Unified Platform for Secure Input & Insights is a full stack web application developed as a final year project. It enables collection, storage, and analysis of feedback data from different categories (students, faculty, visitors, etc.) while ensuring anonymity and data privacy.

This platform incorporates NLP techniques and a Support Vector Machine (SVM) classifier for real-time sentiment analysis of user feedback. Submission trends, smart categorization, and secure input storage are key highlights of this system. The project is inspired by real-world challenges in anonymous communication and decision-making processes.

  • Secure feedback collection via token-based access
  • Sentiment classification using SVM (90%+ accuracy)
  • Live analytics of submission trends
  • Role-based admin and user access
  • Anonymous yet categorized input storage

Start Date: March 2025

Completion Date: April 2025

Keywords: Secure Feedback, NLP, SVM, Anonymity, Smart Data Analysis, Final Year Project

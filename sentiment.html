Project Overview

This Sentiment Analysis project uses NLP techniques to classify feedback as Positive, Negative, or Neutral. Built using SVM, the model achieves ~90% accuracy.

Applications include customer feedback systems, product review analytics, and social media monitoring.

  • Preprocessing: Tokenization, stop-word removal, lemmatization
  • SVM classifier for high precision
  • Real-time sentiment prediction
  • Accuracy metrics and performance insights
